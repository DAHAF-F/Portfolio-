28-Point Self-Audit

Before running this audit framework on client accounts, I ran it on myself first.

Technical SEO Audit GBP Optimization Local Ranking Factors Lahore, PK

The Brief

Before running an audit framework on client accounts, I ran it on my own Business Profile in Lahore first. This case file is a line-by-line technical audit built as a checkpoint list — 28 individual points, rolled up into six broader ranking categories — used to catch weak spots in the methodology itself before it touches real client work.

The Six Ranking Categories

  • Profile completeness & categories — business info, attributes, services, and description accuracy.
  • NAP & citation consistency — name, address, and phone matching across the web.
  • Reviews & reputation signals — volume, recency, response rate, and sentiment.
  • Photos, posts & engagement — how active and visually complete the profile is.
  • Local Pack competitiveness — how the profile stacks up against nearby competitors for target terms.
  • Technical & backend health — duplicate listings, suspension risk, and structured data.

Why It Mattered

Scoring each of the 28 checkpoints against my own profile surfaced blind spots in the audit framework itself — questions that were ambiguous, checks that were redundant, and a few that were missing entirely. That refined framework is what now gets used on client GBP audits, including the Westminster Dental Arts Group case file.
